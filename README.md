# Tailrunner Frontend
[Tailrunner 바로가기](http://tailrunner.run)

## 소개
Tailrunner는 제 개인 포트폴리오 프로젝트입니다. 알고리즘과 자료구조 공부를 하며 Leetcode라는 온라인 알고리즘 문제 풀이 플랫폼을 사용했습니다.
매일 플랫폼을 사용하며 저는 플랫폼 이용자를 넘어 내부 시스템에 관심을 가지게 되었습니다.본 프로젝트는 Leetcode를 Clone Coding 한 프로젝트입니다. Clone Coding이란 실제 서비스를 따라 만들면서 배우는 과정입니다. 
본 Repo는 프로젝트의 Frontend 부분을 구현한 코드를 저장합니다.
